# Redes Neuronales — Fundamentos

Sitio web estático e interactivo sobre fundamentos de redes neuronales, diseñado como material docente en español.  
Combina visualizaciones, explicaciones conceptuales, ecuaciones e interfaces interactivas para introducir temas como perceptrón, funciones de activación, forward pass, backpropagation, descenso de gradiente, caja negra y autoencoders.

## Contenido

El sitio cubre, entre otros, los siguientes temas:

- Neurona artificial y perceptrón
- Funciones de activación: sigmoid, tanh, ReLU, Leaky ReLU y ELU
- Vanishing gradient
- Arquitectura feedforward y forward pass
- Funciones de pérdida
- Backpropagation
- Descenso de gradiente y optimizadores
- Problema de la caja negra en redes neuronales
- Autoencoders:
  - undercomplete
  - denoising
  - overcomplete + sparse

## Enfoque

Este material tiene foco didáctico.  
Combina formulaciones exactas con algunas simplificaciones pedagógicas para facilitar la intuición y la enseñanza introductoria.
